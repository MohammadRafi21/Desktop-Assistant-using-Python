# Desktop-Assistant-using-Python


# How to run:

1. Create a new virtual environment

```bash

conda create -n assistant1 python=3.8

```
assistant1 is the name of the environment you can give whatever you like

2. Check the environment has been created

```bash

conda env list

```

3. Activating the created virtual environment

```bash

conda activate assistant1 ##(name of the environment)

```

4. Check the packages 

```bash

pip list #(to see the version of the python package)

```

5. Install all the packages present in the requirement file

```bash

pip install -r requirements.txt

```
